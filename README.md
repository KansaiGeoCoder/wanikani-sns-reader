# wanikani-sns-reader

Features
WaniKani Integration

Connects via your API key to fetch your current level
Pulls all vocabulary words for your level automatically

Smart Social Media Posts

Generates Japanese social media posts using your level-specific vocabulary
Posts feel natural, like real Twitter/X content
Click any highlighted word to see meanings and readings

Interactive Learning

Hover over vocabulary to see it's clickable
Click words for instant definitions in a clean modal
Refresh button generates new posts with different vocabulary combinations

How to Use

Get your WaniKani API key from WaniKani Settings
Enter it in the app
Start reading Japanese posts at your level!
Click highlighted words to review vocabulary
